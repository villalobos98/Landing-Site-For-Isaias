CSCI-251 Concepts of Parallel and Distributed Systems

Project information

Category: Parallel Programming

In this course, I was taught to write code in C# using the .NET framework to handle HTTP requests and responses as well as concurrency related tasks. The course was structured with 3 projects in mind. In each one, I had to use the concepts learned to develop classes and methods by creating original code to complete my task. Project 1 was invovled with creating a multithreaded program that would mimic the "du" command in Unix based systems. Performance would be assessed by checking the run-time speed of the program. Project 2 was involved with created a Prime Number Generator that would simply find primes concurrently. Project 3 was involved with creating a program(client) that would communicate with a server using HTTPS methods and also utilizing the RSA algorithm as well as Project 2 to send secure messages. In this classs I learned about the following:

  • Understand Parallelism and Concurrency (Concepts)
  • Recognize problems with Parallelism
  • Write Efficient Parallel Programs (C# ForEach)
  • Understaning basics of Network Security and Cloud Computing
  • Debugging Parallel Programs
  • Learn Basics of Routing and Networking

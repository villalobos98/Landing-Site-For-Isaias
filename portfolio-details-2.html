CSCI-250 Concepts of Computer Systems

Project information

Category: Systems Programming

In this course, I was taught to write assembly code based on MIPS architecture. The class was structured so that there were experiments and projects. This was one of the most difficult courses that I had to take. The project that I had to create was the Game of Life, in which, I needed to simulate grass, trees, and wildfires. Depending on the wind condition, either more or less grass/trees caught on fire, and then produced more offspring. In this classs I learned about the following:

  • Instruction Set Architectures (RISK vs CISK)
  • Addressing Memory (Text vs Data Section)
  • Control Flow (Jump Instruction)
  • Switch (Branching)
  • Debugging
  • Implementing Algorithms (Merge Sort)
  • Degital Design (Simulate computer hardware with Logism)
  • Arrays & Pointers

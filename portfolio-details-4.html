Project information

Category: Object-Oriented Programming

In this course, I was taught to write Java code based on Java 9. This class was also similarly structured so that there were homework assignments every week with two major projects. I really enjoyed this class because in this class I learned about algoithms, data structures, object-oriented programming with Java. The project that I had to create was an a small interpreter that could parse tokens, understand tokens, and perform computation based on those tokens. The other major project was a GUI project that was based on a card game, where object-oriented programming came into play.

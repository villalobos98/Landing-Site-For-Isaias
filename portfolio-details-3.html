CSCI-142 Introduction to Python Programming

Project information

Category: Procedural Programming

In this course, I was taught to write python code based on Python 3.5. The class was structured so that there were homework assignments every week with two major projects. I really enjoyed this class because in this class I learned about algoithms, data structures, and Python. The project that I had to create was the a visualization project where I had to use Python and Turtle to create a graph that would show the rate of death within a 50 year time range. I had to organize data and then visualize it. In this class I learned about the following:

  • Introduction to Python, Functions and IDE
  • Parameters, Conditional Statements, Types, Scopes
  • Recursion
  • String, Loops, System I/O
  • Testing & Debugging
  • Python List/Tuples, Searching, Sorting, Complexity
  • Sorting Algorithms
  • Stacks & Queues
  • Iteration vs Recursion
  • Trees
  • Hashing
